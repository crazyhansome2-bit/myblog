# 用 Codex 搭建了属于自己的个人网站

今天做了一件很有纪念意义的事：我用 Codex 搭建并完善了属于自己的个人网站。

它不只是一个能打开的网页，也不只是一个放链接和头像的地方。更像是我给自己准备的一个小据点：可以放作品、放文章、放项目、放照片，也可以慢慢记录游戏、音乐、生活和一些突然冒出来的想法。

以前总觉得“个人网站”听起来像一件很正式、很遥远的事，好像必须先把设计想得足够完整，把内容准备得足够丰富，才能开始动手。但今天真正做下来以后，我反而觉得它最重要的不是一开始就完美，而是先有一个属于自己的空间，然后在之后的日子里不断更新、调整和生长。

## 从一个想法开始

一开始我只是想让网站更像“我自己”一点。

不是模板里的默认文字，不是冷冰冰的占位内容，而是能看出我的兴趣、我的审美、我的项目和我的生活痕迹。于是今天围绕这个目标，我和 Codex 一起一点点改了很多细节：

- 修好了音乐模块，让播放器重新有歌词可以看。
- 把网站头像换成了我自己选的图片。
- 调整了小猫助手的 UI，让它更贴合网站氛围。
- 更新了 GitHub、Gitee、邮箱、QQ、微信等联系方式。
- 在关于我页面里加入了“游戏发烧友”的介绍。
- 把最近做的项目《深坠异想奇境》上传到了项目页，并加入项目文档和架构 PDF。
- 增加了首页宣言文案轮换，让个人卡片里会浮现不同的站点宣言。

这些事情单独看都不算特别大，但组合起来以后，网站的气质就慢慢变了。它开始不只是“一个页面”，而是更像一个正在启动的个人空间。

## 和 Codex 一起搭建的感觉

今天最有意思的地方，是整个过程并不是我一个人在摸索。

我提出想法，Codex 去读项目结构、找文件、改代码、跑构建、提交推送。中间也会遇到一些小问题，比如路径权限、构建警告、组件文案位置不符合预期，或者某个 UI 比想象中偏大。但这些问题都可以边看边改，像是在和一个懂代码的搭档一起装修自己的房间。

我可以很直接地说：

> 这个地方想换成我的头像。  
> 这里的文字想改成黑海岸泰提斯系统。  
> 这个项目背景用我发的图片。  
> 这些宣言不要放在关于页里，要进站弹窗。

然后 Codex 会把这些想法落到具体文件里，变成实际能运行、能部署、能打开的网站功能。

这种感觉挺特别的。它让我觉得建站不再只是“写代码”，更像是把脑子里的画面一点点具象化。

## 今天完成的网站小功能

今天的网站已经有了很多我喜欢的部分。

首页会展示个人信息、文章、随笔、照片墙和音乐播放器。音乐播放时有歌词条，页面背景也会根据我选的图片呈现一种比较柔和的氛围。网站里还有小猫助手、主题切换、全局工具按钮，以及一些轻微的动效。

关于页则更像个人档案。它记录我的介绍、联系方式和活动时间线。现在活动空状态也改成了“黑海岸泰提斯系统暂无活动记录”，这个设定感我很喜欢。之后每次写文章、发随笔或者记录生活，活动页都会留下新的痕迹。

项目页今天也正式放入了《深坠异想奇境》。这是我最近做的项目，所以把它上传到网站里时，会有一种“终于把作品放进展柜”的感觉。项目卡片用了我选的图片做背景，并提供了项目文档和系统架构 PDF 的入口。

还有一个很有仪式感的小功能：首页宣言文案轮换。

现在打开首页时，个人卡片里的简介不再只是固定的一句话，而是会在几段宣言之间慢慢浮现、切换。它们有时像存档点，有时像任务日志，有时又像给自己留的一句提醒。

我很喜欢这种感觉。它不会打断访问者进入网站，但又能让网站在安静的地方多一点自己的语气。

## 网站对我的意义

对我来说，这个网站不只是展示用的。

它更像一个长期存档点。

我可以在这里记录今天做了什么，最近玩了什么游戏，听到了什么歌，做了什么项目，产生了什么想法。很多东西如果不写下来，很快就会散掉。但如果有一个地方能把它们收起来，它们就会变成某种可以回头看的轨迹。

也许以后这个网站会越来越丰富：有更多文章、更多照片、更多项目、更多游戏相关的内容。也许 UI 还会继续改，功能还会继续加，甚至会出现一些现在还没有想到的新栏目。

但今天至少已经完成了第一步。

它上线了，它开始记录了，它已经是我的网站了。

## 写在最后

今天的关键词应该是“开始”。

不是那种轰轰烈烈的开始，而是很具体、很真实的开始：换一张头像，写一段介绍，上传一个项目，修一个播放器，推送一次代码，然后刷新网页，看见自己的想法真的出现在屏幕上。

这感觉还挺好。

希望以后我能经常回来更新它。写文章也好，放项目也好，记录生活也好，都算是在给这个小据点添砖加瓦。

这里会慢慢变成我的个人终端，我的存档点，我在互联网上留下的一小块地方。
